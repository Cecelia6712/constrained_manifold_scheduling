# 约束流形上的导航与进化：酒厂智能排产系统的闭环稳健调控模型

本仓库为数模校赛论文《约束流形上的导航与进化：酒厂智能排产系统的闭环稳健调控模型》的完整配套代码。

---

##  文件列表

```
constrained-manifold-scheduling/
│
├── 附件1.csv                        # 历史排产数据（5348条记录）
├── 附件2.csv                        # 第13周需求（27个产品-酒质组合）
├── 附件3.csv                        # 月度需求（28个产品-酒质组合）
│
├── q1.py                            # 问题一：产能估计
│                                    #   - 收缩分位数估计（k=5）
│                                    #   - CRITIC客观赋权
│                                    #   - Bootstrap自助法（2000次重抽样）
│
├── solve_q2.py                      # 问题二：字典序MILP周排产
│                                    #   - 五阶段字典序优化
│                                    #   - 140个线班（14产线 × 10班次）
│
├── verify_q2.py                     # 问题二：结果验证（26项一致性检查）
│
├── solve_q3.py                      # 问题三：市场均衡月排产
│                                    #   - IIS不可约不可行子系统剪枝
│                                    #   - 原始-对偶同步逼近（10轮迭代）
│                                    #   - 644个线班（14产线 × 46班次）
│
├── verify_q3.py                     # 问题三：结果验证
│
├── problem4_final.py                # 问题四：鲁棒投影导航
│                                    #   - 几何投影导航重调度
│                                    #   - 三级触发机制（静默/微调/紧急）
│                                    #   - 自适应参数校正（β, θ₁, θ₂）
│                                    #   - 5000次Monte Carlo仿真
│
├── results(1).xlsx                  # 问题一输出：产能参数、偏好排序
├── 问题二_字典序MILP排产方案(2).xlsx # 问题二输出：周排产明细、敏感性
├── 问题三_排产结果(1).xlsx          # 问题三输出：均衡排产明细、Pareto方案
└── 第四问.csv                       # 问题四输出：5种策略Monte Carlo对比
```
